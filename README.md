# Student-grade-management-system


**** Project Overview

The Student Grade Management System is a simple Python-based console application designed to store, update, delete, and display student grades using a dictionary. This project demonstrates core Python concepts such as functions, loops, conditionals, and data handling.

The program allows users to interact through a menu-driven interface, making it easy for beginners to understand how CRUD (Create, Read, Update, Delete) operations work.

****Steps to Run the Project

Follow the steps below to execute the program on your system:

1. Install Python

Ensure that Python 3.x is installed on your computer. You can check by running:

python --version

2. Create a Python file

Save the provided code in a file named:

student_grades.py

3. Run the program

Open a terminal or command prompt in the directory where the file is saved and run:

python student_grades.py

4. Use the menu options

You will see a menu like this:

1. Add Student
2. Update Student
3. Delete Student
4. View Students
5. Exit

Enter the corresponding number to perform an action.
